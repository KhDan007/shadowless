# Project Memory

## Core
Direction: Bureau / Forensic Dossier. Warm off-black (#0e0d0b) + paper cream (#e8e2d4). Single signal-amber accent (#d97706 dark / #b45309 light). NEVER reintroduce emerald or the old #10b981.
JetBrains Mono is body. Geist is chrome/UI. Instrument Serif italic for dossier marks only. Inter is banned.
Zero border-radius. Hairline rules in --panel-border. No glow shadows, no scan-sweep, no pulsing. Motion is steps() / instant.
Risk is one hue (amber, lightness-ramped). CRITICAL is the only non-amber (#b91c1c). No green=good.
Dark + Light first-class. Light = ivory paper + ink. Toggle via `light` class on <html>.
Always use semantic tokens. Hex literals like bg-[#0b0e14] in components are legacy debt — migrate when touching a file.
